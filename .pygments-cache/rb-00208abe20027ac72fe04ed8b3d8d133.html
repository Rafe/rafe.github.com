<h1>Sinatra delegation mixin. Mixing this module into an object causes all</h1>

<h1>methods to be delegated to the Sinatra::Application class. Used primarily</h1>

<h1>at the top-level.</h1>

<p>module Delegator #:nodoc:
  def self.delegate(*methods)</p>

<pre><code>methods.each do |method_name|
  define_method(method_name) do |*args, &amp;block|
    return super(*args, &amp;block) if respond_to? method_name
    Delegator.target.send(method_name, *args, &amp;block)
  end
  private method_name
end
</code></pre>

<p>  end</p>

<p>  delegate :get, :patch, :put, :post, :delete, :head, :options, :template, :layout,</p>

<pre><code>       :before, :after, :error, :not_found, :configure, :set, :mime_type,
       :enable, :disable, :use, :development?, :test?, :production?,
       :helpers, :settings
</code></pre>

<p>  class &lt;&lt; self</p>

<pre><code>attr_accessor :target
</code></pre>

<p>  end</p>

<p>  self.target = Application
